# je_ecs_universe_set_frame_deltatime

## 函数签名

```cpp
JE_API void je_ecs_universe_set_frame_deltatime(void* universe, double val);
```

## 描述

设置 Universe 的帧间隔时间。

## 参数

| 参数名 | 类型 | 描述 |
|--------|------|------|
| universe | void* | Universe 实例指针 |
| val | double | 帧间隔时间（秒） |

## 返回值

无
