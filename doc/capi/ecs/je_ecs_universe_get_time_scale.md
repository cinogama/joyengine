# je_ecs_universe_get_time_scale

## 函数签名

```cpp
JE_API double je_ecs_universe_get_time_scale(void* universe);
```

## 描述

## 参数

| 参数名 | 类型 | 描述 |
|--------|------|------|
| universe | void* | Universe 实例指针 |

## 返回值

| 类型 | 描述 |
|------|------|
| double | 时间缩放比例 |
